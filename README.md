# Azure Machine Learning Training - Day 3

## Hands-On Training: Azure ML for Ethiopian Defense University

### Overview
This repository contains all materials for Day 3 of the AI Lab Training program. You will learn how to:
- Connect to Azure Machine Learning workspace
- Train a machine learning model on Azure
- Register and version models
- Deploy models as REST API endpoints
- Test and consume deployed models

---

### Prerequisites

| Item | Status |
|------|--------|
| Azure account with Contributor access | ✅ |
| Access to Azure ML Studio (ml.azure.com) | ✅ |
| Python 3.8+ (for local testing) | Optional |
| GitHub account | Optional |
